== Description ==

Tropical Island serves as a child theme of Blockstarter, inheriting all the features of Blockstarter. The theme features beautiful illustrations and empowers you to take advantage of the latest full site editing experience in WordPress. The theme is suitable for different kind of blogs and portfolio websites like travel blogs, tourist blogs, journey blogs, adventure blogs, summer vacation blogs, travel guides, travel agencies, tour operators and others. We have made sure the theme is compatible with all the popular plugins including WooCommerce and the good old Classic editor. Theme demo: https://tropical-island-demo.nasiothemes.com/
